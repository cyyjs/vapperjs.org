Vapper

A Vue-based server-side rendering framework

Get Started →

Simplicity First

The experience of developing an SSR application with Vapper is like developing a SPA application. No redundant concepts need to learn. Vapper allows you to prefetch data in a very intuitive and consistent way.

Flexible & powerful

Vapper allows you to selectively enable SSR and fine-grain to routing level control. Vapper also allows you to use SSR, Fallback SAP, pre-rendering and more at the same time.

Plugin architecture

Vapper's plugin architecture makes it very extensible. If you've written Vue-cli3 plugins, you'll be familiar with Vapper's plugin authoring. Vapper supports cookie, prerendering, Apollo and more through plugins.
